# HelloNote - Admin Notes Plugin

A simple and elegant WordPress plugin that helps site administrators keep track of important information, reminders, and communicate with each other through a dedicated notes system.

## What is HelloNote?

HelloNote is a lightweight WordPress plugin designed for website administrators who need a quick and easy way to leave notes for themselves or other admins. Think of it as a sticky note board in your WordPress admin area - perfect for tracking tasks, leaving reminders, or sharing important information with your admin team.

## Why Use HelloNote?

- **Quick Access**: Notes appear right on your WordPress dashboard with a convenient widget
- **Team Communication**: Share important information with other administrators
- **Track Important Items**: Keep todos, reminders, and important notes in one place
- **Always There**: Notes stay in your admin area, always accessible when you need them
- **Simple & Clean**: No complicated setup or learning curve - just start writing notes

## Installation

### Easiest Method (Recommended)

1. Download the HelloNote plugin ZIP file
2. In your WordPress admin, go to **Plugins ‚Üí Add New**
3. Click **Upload Plugin** at the top of the page
4. Choose the ZIP file and click **Install Now**
5. Click **Activate Plugin**

That's it! You'll now see an "Admin Notes" widget on your dashboard.

### Manual Installation

1. Upload the `hellonote` folder to `/wp-content/plugins/`
2. Go to **Plugins** in WordPress admin
3. Find "HelloNote" and click **Activate**

## How to Use HelloNote

### Adding a Note

**From the Dashboard:**
1. Go to your WordPress Dashboard (wp-admin)
2. Find the "Admin Notes" widget
3. Type your note in the text area
4. Click "Post Note"

**From the Notes Page:**
1. Go to **Tools ‚Üí HelloNote** in the admin menu
2. Use the "Add New Note" form at the top
3. Type your note and click "Add Note"

### Viewing Notes

- **Dashboard Widget**: See your 5 most recent notes on the dashboard
- **Full Page**: Visit **Tools ‚Üí HelloNote** to see all notes

### Editing a Note

1. Go to **Tools ‚Üí HelloNote**
2. Find the note you want to edit
3. Click the "Edit" button
4. Make your changes and click "Update"

### Deleting a Note

1. Find the note you want to remove (dashboard widget or notes page)
2. Click the trash icon or "Delete" button
3. Confirm the deletion

Any administrator can delete any note - perfect for cleaning up completed tasks.

## Who Created Each Note?

Every note displays:
- The name of the administrator who created it
- How long ago it was posted (e.g., "5 minutes ago", "2 hours ago")

This makes it easy to track who said what and when.

## Requirements

- **WordPress**: Version 5.8 or higher
- **PHP**: Version 7.4 or higher
- **User Role**: Administrator access to use the plugin

## Translation

HelloNote can be translated into any language. The plugin includes:
- Translation template file (`.pot`)
- Support for all translatable strings (including JavaScript)

To translate, use a tool like [Poedit](https://poedit.net/) or [Loco Translate](https://wordpress.org/plugins/loco-translate/).

## Frequently Asked Questions

### Can editors or other user roles see the notes?

No. Only users with administrator capabilities can access and see HelloNote. This keeps your admin communication private.

### What happens to my notes if I deactivate the plugin?

Your notes are safe! The plugin preserves all data when deactivated. You can reactivate it anytime and your notes will still be there.

### Can I use this for customer notes or public content?

HelloNote is designed specifically for administrator-to-administrator communication. It's not meant for customer-facing content or public notes.

### How many notes can I create?

There's no built-in limit. You can create as many notes as you need.

### Will this slow down my website?

No. HelloNote only loads on admin pages and uses efficient database queries. Your website visitors won't be affected at all.

### Can I export my notes?

Export functionality is not currently available but may be added in a future version.

### What if I delete a note by accident?

Deleted notes cannot be recovered, so make sure you want to delete a note before confirming.
